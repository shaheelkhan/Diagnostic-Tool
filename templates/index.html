Diagnostic Tool


This application helps one to check whether a person needs to consult a doctor or not based on his/her condition. Right now this application supports one to check Heart, Liver or Diabetes related conditions.



How to Use

1. Navigate to the health condition that one wants to check
2. Enter the required details
3. Click Submit
4. Based on the entered values,a machine learning algorithm will run in the background and suggest whether the person needs to consult a Physician or not.
